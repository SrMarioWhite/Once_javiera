### Pasos para crear un repositorio en GitHub.

1. **Inicia sesión en GitHub**  
   Ve a [github.com](https://github.com) y accede con tu cuenta.

   2. **Crear un nuevo repositorio**  
      - En la esquina superior derecha, haz clic en el ícono **+**.  
         - Selecciona **New repository**.

         3. **Configurar el repositorio**  
            - **Repository name**: escribe el nombre que quieras darle.  
               - **Description**: opcional, agrega una breve descripción.  
                  - **Visibility**: elige si será **Public** (visible para todos) o **Private** (solo para ti y quienes invites).  
                     - Opcional: marca **Add a README file** para que se cree automáticamente un archivo inicial.

                     4. **Crear el repositorio**  
                        - Haz clic en **Create repository**.  
                           - GitHub te llevará a la página principal de tu nuevo repositorio.

                           5. **Subir archivos desde la web**  
                              - Dentro del repositorio, haz clic en **Add file** → **Upload files**.  
                                 - Arrastra o selecciona los archivos desde tu computadora.  
                                    - Haz clic en **Commit changes** para guardarlos.
